# 动态规划 (DP) 基础通关指南 🧠

动态规划的核心哲学是：**记住过去，推导未来**。

---

## 一、 DP 的三大核心齿轮 (Core Gears) ⚙️

无论题目多复杂，拆解 DP 的步骤永远是这三步：

1.  **定义状态 (`dp` 数组的物理意义)** 明确 `dp[i]` 或 `dp[i][j]` 到底代表什么。例如：“到达第 i 阶的路径数”或“前 i 间房能偷到的最高金额”。
2.  **寻找状态转移方程 (灵魂所在)** 思考“未来”如何由“过去”推导出来。
    * *爬楼梯*：`dp[i] = dp[i-1] + dp[i-2]`
    * *打家劫舍*：`dp[i] = max(dp[i-2] + nums[i], dp[i-1])`
    * *不同路径*：`dp[i][j] = dp[i-1][j] + dp[i][j-1]`
3.  **初始化起点 (宇宙大爆炸)** 确定推导的基石。通常是 `dp[0]`、`dp[1]` 或边界行/列。

---

## 二、 基础模型回顾 🎒

| 模型分类 | 经典题目 | 核心逻辑 | 技巧总结 |
| :--- | :--- | :--- | :--- |
| **斐波那契型** | 爬楼梯 | 当前步 = 上一步 + 上上步 | 状态压缩的入门级 |
| **线性/序列 DP** | 打家劫舍 | 选 vs 不选 (选则不能选相邻) | 哨兵技巧 (虚拟起点 0) |
| **完全背包** | 零钱兑换 | 选哪个 (无限量供应) | 初始化为 `amount + 1` (无穷大) |
| **二维网格** | 不同路径 | 只能向右/向下移动 | 画图制表 (Tabulation) 是最强外挂 |

---

## 三、 高阶心法：空间优化 (Space Optimization) ⚡

* **状态压缩 (Rolling Variables)** 如果 `dp[i]` 只依赖于 `dp[i-1]` 和 `dp[i-2]`，可以用几个变量代替数组，将 $O(N)$ 空间降至 $O(1)$。
* **降维打击 (Dimension Reduction)** 在二维网格中，如果当前行只依赖于上一行，可以用一个一维数组 `dp[j]` 循环更新，将 $O(M 	imes N)$ 空间降至 $O(N)$。

---

## 四、 避坑与调试指南 🚧

* **越界风险**：访问 `dp[i-2]` 或 `dp[j-coin]` 前，务必检查 `i >= 2` 或 `j >= coin`。
* **初始化数值**：求最小值 (`min`) 时初始化为大数；求最大值 (`max`) 时初始化为 0。
* **画图！画图！画图！**：不要在脑子里凭空想象。在纸上画出 DP 表格，手动填前几格，方程自然就出来了。
